# homebrew-tuibookie

Homebrew tap for [tuibookie](https://github.com/orvad/tuibookie) -- a fast, interactive terminal bookmark manager for CLI commands.

## Install

```sh
brew install orvad/tuibookie/tuibookie
```

Or add the tap first:

```sh
brew tap orvad/tuibookie
brew install tuibookie
```

## Update

```sh
brew upgrade tuibookie
```

## Uninstall

```sh
brew uninstall tuibookie
```
